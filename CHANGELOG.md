# Changelog

## [2.0.0] - 2024-03-08
### Added
- Universal LLM Service supporting 12+ providers.
- Interactive CLI Setup Wizard (`civion setup`).
- Bundled Next.js frontend within the Python package.
- Static HTML export support.
- File-based configuration stored in `~/.civion/`.
- Automatic browser launching on start.
- GitHub Actions for auto-publishing and testing.

### Changed
- Rebuilt core configuration system.
- Refactored CLI into modular command structure.
- Updated server to serve static assets for production-ready deployment.
