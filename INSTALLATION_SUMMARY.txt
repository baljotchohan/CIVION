🚀 FIRST TIME INSTALLATION (Step by Step)
════════════════════════════════════════════════════════════════════════════════

STEP 1: Get the code
  - You already have CIVION-main-4 folder or zip

STEP 2: Open terminal/command prompt
  - Windows: PowerShell or Command Prompt
  - Mac: Terminal
  - Linux: Any terminal

STEP 3: Navigate to folder
  cd ~/Downloads/CIVION-main-4
  (or wherever you downloaded/extracted it)

STEP 4: Check Python installed
  python --version
  (must be 3.9 or higher)

STEP 5: Run installer
  Windows:   install.bat
  Mac/Linux: ./install.sh
  Any:       python install.py

STEP 6: Get API key
  Go to: https://console.anthropic.com/
  Create free account
  Generate API key

STEP 7: Edit .env file
  nano .env  (Mac/Linux)
  notepad .env  (Windows)
  Paste your API key

STEP 8: Start server
  python -m civion.api.server

STEP 9: Test in another terminal
  civion goal create "Test"
  civion goal list

✓ DONE! System working

════════════════════════════════════════════════════════════════════════════════

⚠️ IF SOMETHING FAILS
════════════════════════════════════════════════════════════════════════════════

1. Copy the exact error message
2. Search for it in CIVION_UNIVERSAL_INSTALLATION_GUIDE.md
3. Follow the fix
4. If still fails, check:
   - Python version: python --version (must be 3.9+)
   - Internet connection: needed for pip install
   - Disk space: need ~1GB free
   - Permissions: on Linux/Mac may need sudo

COMMON ISSUES:
  - "python not found" → Install from python.org
  - "pip not found" → Already included in Python install
  - "Port 8000 in use" → Run on different port: API_PORT=8001
  - "API key error" → Get key from console.anthropic.com
