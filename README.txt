⚡ QUICK INSTALLATION (All platforms)
═══════════════════════════════════════════════════════════════════════════════

STEP 1: Copy CIVION-main-4 folder (if not already present)

STEP 2: Open terminal in CIVION-main-4 folder
  Windows: Right-click → Open PowerShell here
  Mac: Terminal → cd /path/to/CIVION-main-4
  Linux: Terminal → cd /path/to/CIVION-main-4

STEP 3: Run ONE of these:
  
  Windows:    install.bat
  Mac/Linux:  chmod +x install.sh && ./install.sh
  Any:        python install.py

STEP 4: Get API key (2 minutes)
  Go to: https://console.anthropic.com/
  Create account → Generate API key

STEP 5: Edit .env file
  Edit CIVION-main-4/.env
  Replace: ANTHROPIC_API_KEY=sk-ant-v1-YOUR-API-KEY-HERE

STEP 6: Start server
  python -m civion.api.server

STEP 7: Test in new terminal
  civion goal create "Test"
